# Project Structure

- `apps/accounts/`: Custom user models and auth logic.
- `apps/clients/`: CRM client and lead management.
- `apps/pipelines/`: Visual pipeline and stage tracking.
- `apps/projects/`: Task and milestone tracking.
- `apps/core/`: Shared base models and utilities.
- `config/`: Django settings and URL configurations.
- `static/`: Global CSS/JS assets.
- `templates/`: Global HTML templates.
