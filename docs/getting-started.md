# Getting Started

Welcome to the SaaS Template Marketplace. This platform is designed to host and sell professional Django SaaS boilerplates.

## Overview

The marketplace consists of two primary layers:
1. **Marketplace Platform (Layer 1):** The central hub for cataloging, purchasing, and managing templates.
2. **SaaS Template Boilerplates (Layer 2):** Standalone Django projects that are delivered to customers upon purchase.

## First Steps

1. **Environment Setup:** Follow the [Local Development Guide](local-development.md) to set up your environment using Docker.
2. **Explore the Catalog:** Browse the `templates_catalog` app to see how templates are structured.
3. **Understand the Architecture:** Read the [Project Structure](project-structure.md) to understand the separation between the platform and the templates.

## Key Concepts

- **Organizations:** Users belong to organizations/workspaces.
- **Licenses:** Each purchase grants a license for a specific template.
- **Delivery:** Initial delivery is manual, with plans for automated repository generation.
