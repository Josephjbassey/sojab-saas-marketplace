# CMS Guide

The marketplace uses Wagtail CMS to manage dynamic content.

## Wagtail Integration

- **Home Page:** Configurable marketing site.
- **FAQ:** Dynamic FAQ management.
- **Template Pages:** Some aspects of the template catalog can be enriched with CMS blocks.

## Content Management for Templates

Templates are designed to be "CMS-enabled," meaning they can pull settings or content from a centralized or local CMS instance.
- Layer 1 CMS: Manages the marketplace content.
- Layer 2 CMS: Each template can optionally include its own Wagtail integration for end-user content management.
